Conditionally executing instructions

Introduction

A conditional statement controls the order that statements are executed. A program can have more than one path of execution. The statements in a program will only execute if the condition is true.

Conditions Berta the Robot can test: 


						frontIsClear();
						facingNorth();
						facingSouth();
						facingEast();
						facingWest();

if-then statement


						if (condition) {

Comparison operators

A comparison operator compares two values to each other. Expressions like x > y return true or false.

== x == y Is x equal to y?
!= x != y Is x not equal to y?
> x > y Is x greater than y?
< x < y Is x less than y?
>= x >= y Is x greater than or equal to y?
<= x <= y Is x less than or equal to y?

Grade Calculator Program

in Getting Started folder

Instructions

Define a function named calculateGrade() that has a parameter named score and prints the letter grade to the console. If the score is greater than or equal to 90, the program should print ‘A’. If the score is greater than or equal to 85, print ‘B’. If the score is greater than or equal to 75, print ‘C’. If the score is greater than or equal to 65, print ‘D’. Otherwise, print ‘F’. For example, calculateGrade(80) prints ‘C’.

Answer


						function calculateGrade(score) {
							if (score >= 90) {
								console.log('A');
							} else if (score >= 85) {
								console.log('B');
							} else if (score >= 75) {
								console.log('C');
							} else if (score >= 65) {
								console.log('D');
							} else {
								console.log('F');
							}
						}

Repeating Instructions

for statements

A for loop is used when you know in advance how many times you want to repeat an instruction. 


						for (initial; condition; step) {
							//statements to be repeated
						}

Example: 


						for (i = 0; i < 5; i++) {
							console.log(i);
						}

while statements

A while loop is more general and is used when you want to execute instructions while some condition is true. 


						while (condition) {
							//statements to be repeated
						}

Example:


						let i = 0;
						while (i < 5) {
							console.log(i);
							i++;
						}

Arrays

Creating an array

An array is a container that can hold a list of items. Each item in the array is called an element. You can create an array using two brackets. 


						let arrayName = []

Example:


						let names = [];

You can also initialize an array with values. For example: 


						let names = ["Berta", "Jane", "John"];

Exercise

Create an array with four song titles and assign it to a variable named songs. Print the array to the console.

Accessing elements in an array

Elements in an array are accessed by their index. The index is the position of the element in the array and begins at 0. 


						arrayName[index]

Example


						let names = ["Berta", "Jane", "John"];

The value of names[0] is Berta.

Exercise

Modifying arrays


						arrayName[index] = value

Example:


						let names = ["Berta", "Jane", "John"];
						names[0] = "Robot";
						console.log(names);	//displays ["Robot", "Jane", "John"]

Objects

Creating an object

An object is a container for storing data that has two parts: a key and a value. A key-value pair is called a property. The key is the name of the property and is used to get the value. You can create an empty object with two curly braces. 


						let person = {};

You can also initialize an object with properties in it. 


						let person = {
							first: "Berta",
							last: "the Robot"
						};

Exercise

Create an object named song that contains the title and artist of your favorite song. Print the object to the console.

Accessing object properties

First method:
objectName.propertyName

Example:
person.first;

Second method:
objectName["propertyName"]

Example:
person["first"];

Exercise

First method:
objectName.propertyName = value

Example:
person.first = "Jane";

Second method:
objectName["propertyName"] = value

Example:
person["first"] = "Jane";

Adding properties to an object

First method:
objectName.newPropertyName = value

Example:
person.age = 99;

Second method:
objectName["newPropertyName"] = value

Example:
person["age"] = 99;

Looping through an object

You use a for-in loop to loop through the properties in an object. 


						for (propertyName in object) {
							console.log(object[propertyName]);
						}

Example:


						for (key in person) {
							console.log(person[key]);
						}

You must use bracket notation when accessing property values in the loop.

Counter Program

in counter folder

Create a program that displays a number on the screen. The number is increased by one when the plus button is clicked. And the number is decreased by one when the minus button is clicked. The count should not go below 0 or above 10.

Instructions

  1. Get the count element and save it to a variable named count. (Hint: use getElementById())
  2. Get the plus button element and save it to a variable named btnPlus.
  3. Add a click event listener to the plus button.
  4. Inside the click listener parse the inner text of the count element into an integer and save to a variable named current. (Hint: use parseInt())
  5. Increment current by one if it is less than 10.
  6. Set the inner text of the count element to current.
  7. Repeat steps 2-6 for the minus button.

Partial Answer


						const count = document.getElementById('count');
						const btnPlus = document.getElementById('btn-plus');
						const btnMinus = document.getElementById('btn-minus');

						btnPlus.addEventListener('click', function() {
						  current = parseInt(count.innerText);
						  if (current < 10) {
						    current++;
						  }
						  count.innerText = current;
						});
